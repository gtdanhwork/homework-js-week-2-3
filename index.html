Bài tập 1: Tính tiền lương nhân viên

Input

  • Lương 1 ngày là 100,000 (Mặc định / Number)
  • Nhập số ngày làm việc (Người dùng nhập / Number)

Process

  • Lập trình tính tổng số lương nhận được
  • Công thức: Lương nhận = Lương 1 ngày * Số ngày làm việc

Output

  • Tổng lương nhận được

Tổng lương nhận được:

0 đ

Bài 2: Tính giá trị trung bình

Input

  • Nhập 5 giá trị số thực (x, y, z, a, b)

Process

  • Tính trung bình 5 số thực đã nhập
  • Công thức:
    avgNumber = (x + y + z + a + b)/5

Output

  • Giá trị trung bình của 5 số avgResult

Giá trị trung bình

0

Bài tập 3: Quy đổi tiền

Input

  • Gía trị USD hiện tại là 23.500 (valuePerUSD / Mặc định)
  • Nhập số USD (inputUSD / Number)

Process

  • Lập trình tính tổng số tiền quy đổi
  • Công thức: convertResult = inputUSD * valuePerUSD

Output

  • Tổng số tiền quy đổi (convertResult / Number)

Tổng quy đổi:

0 đ

Bài tập 4: Tính chu vi, diện tích hình chữ nhật

Input

  • Nhập chiều dài (inputLength / Number)
  • Nhập chiều rộng (inputWidth / Number)

Process

  • Lập trình tính chu vi (diameterResult) và diện tích (areaResult) của hình chữ nhật
  • Công thức chu vi: diameterResult = (inputLength + inputWidth) * 2
  • Công thức diện tích: areaResult = inputLength * inputWidth

Output

  • Kết quả chu vi (diameterResult) và diện tích (areaResult) hình chữ nhật

Kết quả chu vi:

0

Kết quả diện tích:

0

Bài tập 5: Tính tổng hai ký số

Input

  • Nhập số có 2 chữ số (inputValue / Number)

Process

  • Lập trình tính tổng 2 ký số vừa nhập
  • Ví dụ: 12 => 1 + 2 = 3
  • Ví dụ: 67 => 6 + 7 = 13

Output

  • Kết quả tổng 2 ký số digitsSumResult

Kết quả chu vi:

0
